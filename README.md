# Tic Tac Toe Game

This is a simple and interactive Tic Tac Toe game developed using HTML, CSS, and JavaScript. The project focuses on basic web development concepts and provides an enjoyable two-player gaming experience in the browser.

---

## Project Features

- Two-player mode
- Interactive user interface
- Real-time game updates
- Win and draw detection
- Responsive layout
- Easy to use design

---

## Technologies Used

- HTML for structure
- CSS for styling
- JavaScript for game logic
